# Light-Source

## Format

### `.json`

#### Perspective

```jsonc
{
  "type": "perspective",
  "fov": 1.57079, // default: 1.57079 (90deg)
  "z-range": [ 0.01, 10000 ] // default: [ 0.01, 10000 ]
}
```

#### Orthographic

```jsonc
{
  "type": "orthographic",
  "scale": 1.0, // default: 1.0
  "z-range" [ 0.01, 10000 ] // default: [ 0.01, 10000 ]
}
```
